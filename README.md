# 開源小島之書

台灣是個美麗的小島，而台灣這個小而美的產業環境也是特別適合開源發展的地方。開源小島之書收錄許多不同的開源觀念，也歡迎提供不同，甚至矛盾的觀點一起激盪。開放作為起點，放下過去，協作未來。
